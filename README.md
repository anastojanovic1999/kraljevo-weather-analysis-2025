# Kraljevo Weather Analysis (2025)
This project analyzes daily temperature and wind observations for Kraljevo, Serbia, during 2025.

# Data Source
- Meteostat
 
# Tools
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

# Analyses
- Temperature statistics
- Extreme temperature days
- Climate indices
- Monthly temperature trends
- Wind speed statistics
- Monthly wind speed trends
